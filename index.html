Configurar Firebase

    
Cole as credenciais do seu projeto Firebase (Project Settings → Web App)


    

      
Como obter as credenciais:

      1. console.firebase.google.com → seu projeto

      2. ⚙ Configurações do projeto → Seus aplicativos

      3. Clique no ícone </> Web (ou crie um app web)

      4. Copie os valores do bloco firebaseConfig

Configurar Google Sheets

    
Cole a URL do Web App publicado no Google Apps Script


    

      
Como obter a URL (siga exatamente):

      1. Planilha → Extensões → Apps Script

      2. Cole o código do arquivo apps_script_banco.gs (versão 2)

      3. Implantar → Nova implantação — não reutilize a anterior

      4. Tipo: App da Web

      5. Executar como: EU (minha conta)

      6. Quem acessa: Qualquer pessoa (sem login)

      7. Implantar → autorize: "Revisar permissões" → sua conta → "Avançado" → "Acessar (não seguro)" → Permitir

      8. Copie a URL .../exec e cole abaixo

Compartilhar — somente leitura

    
O destinatário poderá ver todos os dados mas não poderá inserir, editar ou excluir.


    

      
Como funciona:

      O arquivo HTML será gerado com o parâmetro ?readonly=1 embutido.

      Ao abrir, o destinatário verá apenas Dashboard, Consolidado e Ranking — sem acesso a inserção ou edição.


      Se você usa Firebase, as credenciais são incluídas automaticamente — o destinatário vê os dados em tempo real sem configurar nada.
